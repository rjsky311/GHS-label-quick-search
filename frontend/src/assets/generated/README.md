# Generated Visual Assets

Project-bound bitmap assets in this folder are generated with Codex built-in image generation and then checked into the app after local QA.

## Current Assets

- `ghs-empty-workflow.webp` - empty-state workflow visual for the first-use screen. Generated on a flat chroma-key background, locally converted to alpha, resized to 960x640, and saved as transparent WebP.

## Guardrails

- Do not replace official GHS pictograms with generated artwork.
- Do not place promotional graphics inside printed labels, signal words, H/P-code content, or hazard summaries.
- Keep generated visuals decorative or workflow-oriented. Safety interpretation still depends on PubChem references, SDS, supplier labels, and local regulations.
