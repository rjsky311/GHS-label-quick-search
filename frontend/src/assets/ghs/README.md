# GHS Pictogram Assets

These SVG files are project-local mirrors of the standard GHS pictograms served
by PubChem at `https://pubchem.ncbi.nlm.nih.gov/images/ghs/GHSXX.svg`.

They are checked in so result rows, detail views, live print previews, and print
iframes do not depend on third-party image availability at runtime.

Do not replace these with AI-generated or decorative artwork. GHS pictograms are
safety identifiers, so edits should preserve the standard red diamond, black
symbol, proportions, and transparent SVG format.
